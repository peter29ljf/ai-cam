手部监控系统

本地运行模式

📋 使用说明

由于浏览器安全限制，在非 localhost 环境下无法访问摄像头。本页面提供了一个替代方案，使用本地 Python 脚本直接运行手部监控功能。

提示： 这种方式更稳定且性能更好，适合于需要长时间监控或采集大量图像的场景。

⚙️ 操作步骤

1

确保您的电脑已安装 Python 环境（Python 3.7+）

🔑 远程访问用户注意：

通过IP地址访问时无法直接使用摄像头（浏览器安全限制）。请直接在本地运行Python脚本，效果更好且无限制。

2

安装必要的依赖包：

pip install opencv-python mediapipe
✓ 复制成功
3

在终端中执行以下命令启动监控程序：

python3 flip_monitor_mediapipe.py --max-count 10 --shots-dir mp_shots
✓ 复制成功
4

程序启动后，将自动监控手部动作：

  • 当检测到手出现时，进入准备状态
  • 当手消失后，延迟 0.5 秒进行截图
  • 截图会保存到 mp_shots 目录下
  • 系统会发出提示音表示截图成功
5

修改参数（可选）：

  • --max-count：设置最大截图数量
  • --shots-dir：设置截图保存目录
  • --min-detect-conf：设置检测置信度阈值（0-1）

🖼️ 实际效果说明

程序运行时会显示摄像头画面，当检测到手部时会显示手部骨架，截图后会自动保存。

使用建议：
  • 相机画面保持稳定，避免晃动
  • 保持光线充足，提高检测准确率
  • 把手完全移出画面才会触发截图
  • 页面翻动速度不要太快，每次等待提示音后再翻下一页

运行程序后，请查看 mp_shots 目录获取实际截图
